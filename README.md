# MONOLIT - Premium Construction Company Website

## 🏗️ Overview

MONOLIT is a premium, fully functional construction company website featuring a modern black and yellow color scheme, advanced animations, and comprehensive functionality.

## ✨ Features

### 🎨 Design & Aesthetics
- **Color Scheme**: Black (#0A0A0A) + Yellow (#FFD700) - Modern, aggressive yet elegant
- **Custom Logo**: Animated SVG star icon with rotating effect
- **Responsive Design**: Fully responsive on mobile, tablet, and desktop
- **Professional Layout**: Clean, modern UI with premium feel

### 📄 Pages & Sections
1. **Hero Section** - Eye-catching introduction with CTA buttons
2. **About Section** - Company mission, history, and animated statistics
3. **Team Section** - Leadership profiles with photos and descriptions
4. **Projects Gallery** - 6+ high-quality construction images with:
   - Filter functionality (All, Residential, Commercial, Industrial)
   - Lightbox modal with detailed project information
   - Smooth hover animations
5. **Services** - 6 service cards with icons and descriptions
6. **Contact** - Fully functional contact form + contact information + map placeholder
7. **Footer** - Social media links, company info, copyright

## 🎨 Customization

### Colors
Edit CSS variables in the `<style>` section:
```css
:root {
    --primary-black: #0A0A0A;
    --primary-yellow: #FFD700;
    /* ... other colors ... */
}
```

### Content
Edit data attributes in HTML elements:
```html
<element data-en="English text" data-ru="Русский текст">English text</element>
```

### Gallery Items
Modify the `galleryData` array in JavaScript section to add/edit projects
